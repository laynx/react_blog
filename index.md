## Install React, the easy way.

First, [install Node JS](https://nodejs.org/en/). Then run the commands below:

~~~html
$ npx create-react-app my-app
$ cd my-app
$ npm start
~~~
Basically just configure things like **Webpack** and **Babel** for you.
